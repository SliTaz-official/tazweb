Documentación de Tazweb

SliTaz Web Browser

TazWeb es un navegador web radicalmente simple y muy ligero, que proporciona una sola ventana con una pequeña barra de herramientas y un menú contextual, pero sin ningún tipo de menú o pestañas. La navegación se realiza con el botón derecho del mouse en cualquier lugar de la ventana del navegador o desde la barra de herramientas. También puede manejar los marcadores en una página personal, proporcionar un modo de vista del origen y funciones de zoom y de impresión. TazWeb le permite buscar rápidamente en la web a través de una entrada de búsqueda y en Google. Este archivo de documentación siempre es accesible desde el lado derecho de la barra de herramientas.

El navegador es muy rápido y pequeño, el binario es de 24 Kb y utiliza el motor Webkit para la renderización de páginas y por lo tanto es compatible con todos los estándares web, así como animaciones Flash, Javascript y videos.

La Historia

El navegador web SliTaz fue creado para las necesidades de TazPanel que es la administración SliTaz y el panel de administrador de paquetes y se hace con las últimas tecnologías como XHTML 5 y CSS 3 para la interfaz gráfica de usuario. El contenido es alimentado por los scripts CGI y el servidor web preferido es httpd Busybox.

Seguridad y datos personales

TazWeb es un navegador muy seguro y no almacena ningún tipo de datos personales, por consiguiente no hay sesiones, cookies ni archivos de historial. Por lo tanto la configuración de sitios web no se guardara y se tendrá que introducir la información de acceso cada vez que reinicie el navegador (por el modo de acceso, desde los sitios web pueden realizar un seguimiento mediante cookies!).

Mi Página de Inicio y Favoritos

TazWeb maneja una página personal que se encuentra en su directorio de configuración Freedesktop estándar, la ruta es: $HOME/.config/tazweb/home.html. Que le permite tener una página personalizada con marcadores y otros contenidos. La hoja de estilo CSS se define en $HOME/.config/tazweb/style.css.

El archivo es un documento simple con XHTML 5, por defecto, una lista de enlaces y una entrada de busqueda. Puede editar este archivo con su editor favorito y añadir contenidos, tales como entradas de favoritos o cambiar a la apariencia que desee. En el primer inicio TazWeb va a crear y copiar su página personal y hoja de estilo a su directorio de configuración.

Para añadir un poco de vida a sus páginas usted puede utilizar su propio reproductor Javascript o algun otro de un sitio web de terceros, como los famosos Widget de Twitter como se ve en la página web principal de SliTaz.

Los marcadores pueden ser manejados por TazWeb y usted puede agregar fácilmente un marcador a su página principal con el menú contextual del botón derecho: "Añadir un marcador". Para funcionar TazWeb utiliza marcas en el documento home.html, este marcado no debe ser removido si usted desea agregar marcadores de trabajo, ya que cada nuevo elemento se añade justo antes de la línea: <!-- end:bookmarks -->

Nota: Usted encontrará el home.html por defecto en /usr/share/tazweb y puede obtener una copia original con los siguientes comandos en la terminal: 

$ cp -r /usr/share/tazweb $HOME/.config/tazweb

La Pagina de Inicio y una URL como argumento

La página de inicio por defecto se ha definido en el código fuente y no se puede cambiar y es el valor predeterminado de SliTaz WebHome, pero TazWeb acepta una URL como primer argumento, por ejemplo, puede ejecutar: 

$ tazweb http://www.slitaz.org/

De este modo, por ejemplo, puede crear su propia página web en un documento XHTML e iniciar tazweb con un archivo estándar de escritorio o desde la línea de comandos: Archivo tazweb://$HOME/.config/tazweb/home.html.

Si desea una página de inicio personalizada para su sistema Live CD o para todos los usuarios del sistema, puede crear su propio paquete de SliTaz-configs o editar el archivo xHTML que se encuentra en /usr/share/webhome/index.html. Si lo modifica, asegúrese de bloquear SliTaz-configs o hacer una copia de seguridad, ya que será reemplazado con un paquete de actualización SliTaz-configs.

Cuando se inicia TazWeb desde la línea de comandos con una URL como argumento,tiene que poner http:// luego TazWeb va a transformar las direcciones URL automaticamente por usted.

URL y la entrada de búsqueda

La entrada URL en la barra de herramientas le permite introducir una dirección de Internet y muestra la ubicación actual sitio web. Usted puede utilizar la selección, copiar y pegar en la entrada de URL sin necesidad de poner http:// antes de la dirección, TazWeb va a transformar la URL por usted. Para activar una nueva dirección URL debe presionar ENTER.

La entrada de búsqueda le permite buscar en la web en cualquier momento a través de Google. Sólo tienes que introducir la consulta y presionar ENTER o haga clic en el icono de entrada para validar su solicitud.

Aplicaciones Web

El navegador TazWeb es perfecto para poner en marcha las aplicaciones web a través de un archivo de escritorio simple. Una vez creados con un editor de texto simple, el archivo de escritorio puede estar en el escritorio con un bonito icono o en el menú estándar de SliTaz. Ejemplo de un lanzador de aplicaciones web que se podría nombrar scn.desktop 

[Desktop Entry]
Type=Application
Name=SliTaz SCN
Icon=slitaz-menu
Exec=tazweb http://scn.slitaz.org/

Errores y Desarrollo

TazWeb es un proyecto de código abierto bajo licencia BSD y todos los recursos son proporcionados por SliTaz GNU/Linux, se utiliza un repositorio de Mercurial para la gestión de código, la discusión y el apoyo se realiza a través del foro de la comunidad SliTaz. Si le faltan las funciones de TazWeb o encuentra algún error. Por favor, comunique el problema a nuestro centro de resúmen de errores en: bugs.slitaz.org

Si quieres ayudar y participar en el desarrollo, puede empezar por clonar el repositorio TazWeb y luego enviar el parche, mejora de la documentación o tu trabajo a la lista de mailing de SliTaz en la SCN. Para clonar o navegar en el repositorio TazWeb Hg: 

$ hg clone http://hg.slitaz.org/tazweb
