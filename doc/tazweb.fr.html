Documentation TazWeb

Le navigateur Web de SliTaz

Tazweb est un navigateur radicalement simple et léger avec une seule fenêtre sans menus ni onglets, mais une barre d'outils très simple. La navigation se fait avec le clic droit de la souris, à n'importe quel endroit de la fanêtre ou de la barre d'outils. Le mode plain écran et cette documentation sont également accessible sur le côté droit de la barre d'outils.

Le Navigateur Web SliTaz a été crée pour les besoins de TazPanl qui est l'outil d'aministration et de gestion de paquets pour SliTaz. L'interface utilisateur utilise les dernières technologies comme XHTML 5 et CSS 3. Le contenu est alimenté par des scripts CGI et utilise le serveur HTTP de Busybox.

Le navigateur est léger, très léger, le binaire pèse à peu prêt 12Ko il utilise Webkit, comme moteur de rendu HTML, qui supporte les derniers standarts du Web telque le Javascript.

Page d'acceuil et URL

La page d'acceuil est définie dans le code source et ne peut être modifiée. Il s'agit de la page d'acceuil "SliTaz WebHome".
Mais Tazweb accepte ausi les URLs comme arguments. Par exemple http://www.slitaz.org.
De cette façon vous pouvez créer votre propre page d'acceuil en XHTML et démarrer Tazweb avec un fichier .desktop ou depuis la ligne de commande.
Comme ceci:

tazweb file://$HOME/fichier.html

Si vous souhaiter personnaliser la page d'acceuil de votre Live CD ou des autres utilisateurs, vous pouvez créer votre propre paquet slitaz-configs ou éditer le fichier /usr/share/webhome/index.html.
Si vous modifiez ce fichier, assurez-vous d'en faire une copie ou bloquez la mise-à-jour du paquet slitaz-configs car vos modifications seront écrassées si vous faîte une mise à jour des paquets.

Applications Web

Le navigateur TazWeb est parfait pour lancer des applications via un fichier .desktop. Une fois crée avec un simple éditeur de texts, le fichier .desktop peut être posé sur le bureau ou dans le menu avec une jolie îcone. Voici un expemple de lanceur pour une application Web: 

[Desktop Entry]
Type=Application
Name=SliTaz SCN
Icon=slitaz-menu
Exec=tazweb http://scn.slitaz.org/
